# 🚀 Student Marks Prediction

This is a simple Machine Learning project to predict student marks based on study hours.

---

## 📊 Project Overview
- Predicts marks using study hours
- Uses Linear Regression algorithm
- Visualizes data using scatter plot

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

---

## 📁 Project Files
- analysis.ipynb → Main code
- data.csv → Dataset

---

## 📈 Output
- Scatter plot showing relationship between hours & marks
- Predicts marks for given study hours

---

## 🎯 Example
Input: 9 hours  
Output: ~95 marks
